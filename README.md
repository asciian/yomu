## 特徴

- 標準でオフライン英和辞書を内蔵
- 曖昧検索に対応

## インストール方法

[リリースページ]()よりWindows版/Linux版をダウンロードしてください。

## 使い方

調べたい英単語をドラックあるいはダブルクリックすることで、
単語の意味を単語周辺に表示します。

## 参考URL

- [EJDict](https://github.com/kujirahand/EJDict)
